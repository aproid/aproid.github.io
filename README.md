# aproid.github.io
Tech blog of the developer called Aproid

## Requirement
* Ruby 2.7 or later
* Ruby Bundler 2.1 or later

## Build
#### Serve Application
```
> cd PROJECT_FOLDER
> bundle exec jekyll serve --open-url --livereload --drafts
```
#### Serve Application (Short-handed)
```
> cd PROJECT_FOLDER
> bundle exec jekyll serve -o -l -D
```
